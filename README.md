## ğŸ”§ Customization

### Update Personal Information
Edit the HTML content in `index.html`:
- Hero section: Update name, title, and description
- About section: Modify bio and info cards
- Skills section: Add/remove skills and categories
- Projects section: Update project details
- Contact section: Change contact information

### Change Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --bg-primary: #0a0a0f;
    --accent-cyan: #00d9ff;
    /* ... other colors */
}
```

### Modify Animations
Adjust animation settings in `js/script.js`:
- Particle count: `numberOfParticles`
- Typewriter speed: Change timeout value in `typeWriter()`
- Scroll animation threshold: Modify `observerOptions`
